# Article Outline

## Title

代码正在贬值，业务定义正在暴涨

## Positioning

- high-controversy, high-signal article
- combines industry observation, historical reflection, and methodology proposal
- aimed at developers, architects, and AI coding practitioners

## Core Claims

1. AI is rapidly reducing the cost of implementation
2. complexity is not disappearing, it is shifting upward into business definition
3. traditional low-code identified the right problem but used the wrong method
4. future leverage lies in definition, not implementation

## Structure

### 1. Opening (anxiety + hook)
- AI can already generate large amounts of code quickly
- developers begin to question their future role
- introduce the core tension

### 2. Explosion claim
- code is becoming cheaper
- introduce the controversial claim

### 3. Reversal
- complexity has not disappeared
- it has shifted from implementation to definition

### 4. Historical perspective
- traditional development
- low-code attempt and limitations

### 5. New paradigm
- primitive semantics
- rule + case + feedback
- draft / preview / release
- session / branch / PR

### 6. Final conclusion
- software development is shifting from implementation systems to definition systems

## Tone

- assertive but reasoned
- provocative but not empty
- grounded in real system design thinking
