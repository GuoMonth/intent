# Decision 0001 - Platform Positioning

## Context

We explored building:

- ERP system
- low-code platform
- AI-assisted system

## Decision

We are building:

> An AI-native business application runtime

NOT:

- a traditional ERP
- a low-code UI builder
- a CRUD backend generator

---

## Implications

- AI is first-class actor
- platform must expose machine-readable primitives
- system must be composable and guarded
- business domains come from plugins

---

## Consequences

- requires strong runtime design
- requires clear primitive model
- requires structured plugin system
- requires AI-friendly error system
